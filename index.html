Project Prospect

  1. Introduction

Many recording devices, like the DJI Mini 4 Pro drone, often split recorded video into multiple clips when the file exceeds a predetermined, uncustomizable limit. For hobbyists and casual users, combining these clips into a single video is inconvenient and time-consuming. This task often requires a bulky and complex video editing software  (Davinci Resolve 19 [2.29GB], Premiere Pro [$263.88yr/8GB]). Furthermore, the only current open-source solutions available are Command Line Interface (CLI) applications or projects that require extensive programming knowledge or understanding of video processing, which are inaccessible for the majority of casual users.

  1. Objectives

This project aims to create a simple, user-friendly application that allows users to easily append their video clips into a single continuous video, without the need for extensive editing knowledge or software. The primary objective is to provide a user-friendly solution for merging clips into one seamless video file. The application is designed to be efficient, and cater to users who want to quickly combine videos without the need to download large complex video editing software or learn complex video editing knowledge.

  1. Target Customers

With the simplicity of our application, and the efficiency of its task, our inferred target customers are as follows:

  • Hobbyist Drone Users: Individuals who use drones like the DJI Mavic 4 Pro for recreational video recording.
  • Casual Videographers: Users who record videos with various devices and need a simple tool to merge clips.
  • Social media content creators: Individuals who create content for platforms like YouTube, Instagram, and Tiktok, and require a quick way to combine clips for posting.
  • Amateur filmmakers: Beginners in filmmaking who need a straightforward solution to merge clips without the complexities of professional editing software.
  1. Value Proposition

This application is necessary because it addresses and solves the specific need of merging video clips efficiently without requiring advanced technical skills or extensive software. It stands out by offering the following:

  • Simplicity: The application is focused solely on appending video clips, avoiding the complexity of full-featured editing software.
  • Speed: The application results in fast processing times for merging clips, saving users time and effort.
  • Accessibility: The application will contain an easy-to-use interface that caters to non-professional users, making video merging quick and straightforward.
  • Filling a Gap: The application is unlike current open-source options that are either command line-based or require coding skills for initial setup, this application provides a simple, graphical interface that any user can operate without prior video editing or software development experience.
  • Unique Features: The application may contain automatic clip detection and merging, support for high-resolution videos, and a minimalistic user interface that enhances user experience.
  1. Application, Features, and Description

Minimum Viable Product:

  • Import Functionality: Users can select multiple video clips from their device, supporting various video formats and resolutions.
  • Merge Functionality: The application will merge the imported clips into a single continuous video file.
  • Export Functionality: Users can save the merged video in different formats and resolutions, with the ability to choose the export destination.
  • Graphical Interface: A simple graphical user-interface for importing clips and an easy to use menu for merging and exporting videos. The focus will be on a clean, user-friendly design that requires minimal user input and provides a seamless merging experience.
